Differential Analysis Info

Enter the range of numbers to signify which samples are in which group. You can use '-' (e.g. 2-10) for an inclusive range and commas to add numbers. For example 1,2,4-7 will include [1,2,4,5,6,7] for one group. The gini information gain allows you to control which genes get selected. A high gini value can bias groups that have more cells while a low gini value can include less significant genes
